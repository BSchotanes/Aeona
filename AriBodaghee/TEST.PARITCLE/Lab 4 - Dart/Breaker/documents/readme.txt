Ari Bodaghee
============


Controls:
---------

A		Move Player Left:
D		Move Player Right:
S		Quit to Game Over Screen:

\/(arrow key)	Remove Aim Shot/ Deactivate Shot:

/\ (arrow key)	Aim Shot / Activate Shot:

< (arrow key)	Angle Shot to Left:
> (arrow Key	Angle Shot to Right:

SPACE_BAR	Shoot Ball:

P		Toggle Pause:



List of Events:
---------------
+ Collision
+ Shoot


List of Messages:
-----------------
+ CreateAsteroid
+ CreateDart

- DestroyAsteroid
- DestroyDart
